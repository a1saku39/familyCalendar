# 📝 Family Calendar - 更新履歴

## 🎉 最新の機能追加 (2026-02-01)

### ✨ 新機能

#### 1. **連続する予定の入力機能**
- 複数日にわたる予定を一度に登録できるようになりました
- 例: 「2月3日〜2月7日まで出張」のような連続した予定を簡単に入力可能

**使い方:**
1. 予定追加モーダルを開く
2. 「連続する予定」チェックボックスをON
3. 開始日と終了日を選択
4. タイトルやカテゴリーを入力して保存
5. 指定した期間の全ての日に同じ予定が自動的に登録されます

**メリット:**
- 長期の出張、旅行、イベントなどを一度に登録
- 入力の手間を大幅に削減
- 各日付に個別のイベントとして保存されるため、後から個別に編集・削除も可能

#### 2. **スマホでの入力体験の改善**
- モーダルを閉じた時に画面サイズが元に戻るように修正
- キーボードが自動的に閉じるように改善
- スクロール位置が自動的にリセット

**改善内容:**
- 入力フィールドにフォーカスした時にキーボードが表示されて画面が拡大
- モーダルを閉じると、自動的に元の画面サイズに戻る
- より快適なスマホでの操作体験

## 🔄 既にGitHub Pagesで公開している場合の更新方法

### 方法1: GitHubのWebインターフェースで更新

1. **GitHubリポジトリにアクセス**
2. **更新したいファイルをクリック**（例: `app.js`）
3. **鉛筆アイコン（Edit）をクリック**
4. **ローカルの最新ファイルの内容をコピー&ペースト**
5. **「Commit changes」をクリック**
6. **同じ手順で `index.html` と `styles.css` も更新**

### 方法2: ファイルを再アップロード

1. **GitHubリポジトリにアクセス**
2. **「Add file」→「Upload files」をクリック**
3. **更新した3つのファイルをドラッグ&ドロップ**
   - `index.html`
   - `app.js`
   - `styles.css`
4. **「Commit changes」をクリック**

※数分後に自動的に反映されます

## 🧪 動作確認

### 連続予定の入力テスト

1. カレンダーを開く
2. 「予定を追加」ボタンをクリック
3. 「連続する予定」にチェック
4. 開始日: 2026-02-03
5. 終了日: 2026-02-07
6. タイトル: 「出張」
7. カテゴリー: 「仕事」
8. 保存

→ 2月3日〜7日の5日間に「出張」という予定が登録されることを確認

### スマホでの動作確認

1. スマホでカレンダーを開く
2. 予定を追加
3. 入力フィールドをタップしてキーボードを表示
4. モーダルを閉じる
5. 画面が元のサイズに戻ることを確認
